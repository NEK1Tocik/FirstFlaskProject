Русский язык:
Это простой проект на Flask с базой данных и простым функционалом. 
Визуальная часть выполнена с помощью Bootstrap.
Файл app.py - само приложение.
Папка (директория) templates - шаблоны ( .html файлы).
Папка (директория) static - статические файлы, в которых находится папка Img с изображением и папка CSS (с .сss файлами).

English language:
This is a simple Flask project with a database and basic functionality.
The visual component is built using Bootstrap.
The app.py file is the application itself.
The templates folder (directory) contains templates (.html files).
The static folder (directory) contains static files, including an img folder with images and a css folder (with .css files).
